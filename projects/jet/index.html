Jet Slalom Clone

Description

This is a clone of the Jet Slalom java applet arcade game.

Goals

  • Honor Jet Slalom with a clone.
  • Try OpenGL 3.2 and GLSL 1.5.
  • Try Java Web Start Deployment.

Dependencies

  • The graphics card used must support OpenGL 3.2 and GLSL 1.5.
  • Compiled with Java 1.6.
  • JOGL library.
  • Colt library.
